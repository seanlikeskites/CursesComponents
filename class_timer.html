Detailed Description

A timer class which calls a given function periodically.

Classes which inherit this class will have their implementation timerCallback() called periodically.

Starts the timer such that timerCallback() will be called periodically using the provided period.

Parameters

If the timer is currently stopped this involves the starting of a new thread.

void Timer::pauseTimer ( )

Causes the timer thread to busy sleep. The timerCallback() will no longer get called by the thread will continue to run in the background.
